# Reproductor de Música en Electron

Un reproductor de música de escritorio construido con HTML, Tailwind CSS, JavaScript y empaquetado con Electron.

## Vista previa
![Vista previa](https://github.com/user-attachments/assets/b94f48ff-e59e-4bee-8da5-7f4530c06e6e)

## Características

* **Reproducción Fluida:** Interfaz que resalta dinámicamente la canción en reproducción.
* **Modo Aleatorio:** Utiliza el algoritmo de **Fisher-Yates** para mezclar las canciones.
* **Modo Bucle:** Repite la canción sin límites.
* **Navegación Eficiente:** Botones de siguiente y atrás.
* **Gestor de Carpetas:** Selecciona cualquier carpeta y el reproductor cargará automáticamente todas las canciones.
* **Control Total:** Barra de progreso interactiva y control de volumen integrado.

## Cómo descargar e instalar

1. Ve a la sección de [Releases](https://github.com/DevOscarLuisV/musica/releases/tag/v1.0.3) de este repositorio.
2. Descarga el archivo más reciente llamado `Musica.Setup.1.0.3.exe`.
3. Haz doble clic en el instalador y sigue las instrucciones en pantalla.
4. ¡Abre la app, elige tu carpeta de música y disfruta!
